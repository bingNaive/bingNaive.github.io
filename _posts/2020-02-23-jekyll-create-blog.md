## 安装ruby和jekyll
安装jekyll需要ruby版本2.4.0以上,macOS Catalina 10.15随附ruby 2.6.3,如果您正在运行以前到macOS系统，则必须安装较新版本Ruby,参考[文档](https://jekyllrb.com/docs/installation/macos/)
{% highlight code %}
#查看ruby版本
ruby -v
{% endhighlight %}
安装Jekyll和bundler gems
{% highlight code %}
gem install jekyll bundler
{% endhighlight %}
安装完成之后进入clone下来到git项目中，输入命令
{% highlight code %}
#--force用于当前目录非空时，不会覆盖掉
jekyll new . --force
{% endhighlight %}
本地运行jekyll网站
{% highlight code %}
jekyll serve
{% endhighlight %}
此时可以在浏览器输入[localhost:4000](https://localhost:4000)查看自己的页面<br/>
## [jekyll主题与插件](https://jekyllrb.com/resources/) 
安装完成jekyll后可以自己修改页面样式，但更简单的方法是直接下载别人的主题<br/>
### 主题<br/>
[jamstackthemes.dev](https://jamstackthemes.dev/ssg/jekyll/)<br/>
[jekyllthemes.org](http://jekyllthemes.org/)<br/>
[jekyllthemes.io](https://jekyllthemes.io/)<br/>
主题下载完成之后可以直接将其直接拷贝到自己到项目中运行查看，需要注意的是下载别人的主题启动时有可能失败，这是因为版本和依赖问题，所以还需要和Bundler一起使用<br/>
打开终端进入项目中进行以下步骤
{% highlight code %}
#创建bundler项目，通过创建一个空的Gemfile
bundle init
#配置Bundler
bundle install --path vendor/bundle
#添加Jekyll
bundle add jekyll
创建Jekyll Scaffold
bundle exec jekyll new --force --skip-bundle .
bundle install
#启动
bundle exec jekyll serve
{% endhighlight %}
现在应该可以启动成功，可以打开浏览器查看，编写属于自己的个人网站了
最后一步，提交git,因为jekyll和bundle有一些自动生成的文件和特定于用户或平台的信息，所以需要加入.gitignore忽略文件
{% highlight code %}
# Ignore metadata generated by Jekyll
 _site/
.sass-cache/
.jekyll-cache/
.jekyll-metadata
# Ignore folders generated by Bundler
.bundle/
vendor/
{% endhighlight %}
